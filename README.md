# Spring-Boot - banking-core

## Pre-requisitos tener instalado
1. Instalar JDK 17
2. Plugin maven
3. Base datos memoria H2
4. Compilar mvn clean install
5. Ejecutar mvn spring-boot:run / docker-compose up --build

## Antecedentes
Proyecto Reto Neoris.
